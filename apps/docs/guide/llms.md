# LLMs & AI Context

Esta página contiene el contexto completo del proyecto en formato `llms.txt` — un estándar emergente para que los modelos de lenguaje (LLMs) y agentes de IA entiendan la estructura y convenciones de un repositorio sin necesidad de explorar todos los archivos.

Puedes usar este contenido directamente en cualquier chat con IA, pegarlo como contexto inicial, o referenciar el archivo `llms.txt` en la raíz del repositorio.

## ¿Para qué sirve?

Cuando trabajas con un agente de IA (Claude, Kiro, Cursor, Copilot, etc.) en este proyecto, puedes darle este contexto para que:

- Sepa qué componentes existen y cómo se llaman
- Entienda la arquitectura CSS de dos capas sin leer todos los archivos
- Conozca los tokens `--hp-*` disponibles
- Siga los patrones correctos de `HeadlessElement` desde el primer mensaje
- No invente convenciones que no existen en el proyecto
